# node-calculate-fee


To install jest:

with npm
```
npm install
```

with yarn

```
yarn
```
before run:

> You can put your valid array of objects in text.json or add another json file and type its name when the program ask you to enter a name 

To run the program:

with npm
```
npm run start
```

with yarn


```
yarn start
```

To run test:

with npm
```
npm run test
```

with yarn


```
yarn test
```
